Declaración de accesibilidad

Declaración de accesibilidad y política de inclusión para el sitio web de Divinci AI

Declaración de accesibilidad

Divinci AI está comprometido con garantizar la accesibilidad digital para personas con discapacidades. Estamos mejorando continuamente la experiencia del usuario para todos y aplicando los estándares de accesibilidad relevantes.

Nuestro compromiso

Estamos dedicados a proporcionar una experiencia inclusiva para todos los usuarios, independientemente de su capacidad o tecnología. Nuestro sitio web tiene como objetivo cumplir con las Pautas de Accesibilidad al Contenido Web (WCAG) 2.1 en el nivel AA.

Características de accesibilidad

Navegación y estructura

  • Enlaces de salto de navegación: Acceso rápido a las áreas de contenido principal
  • Estructura clara de encabezados: Jerarquía lógica de H1 a H6 para lectores de pantalla
  • Regiones de referencia: Navegación, contenido principal y áreas de pie de página debidamente marcadas
  • Navegación consistente: Diseño y estructura de menú predecibles en todas las páginas

Diseño visual

  • Alto contraste: Combinaciones de colores que cumplen con los estándares WCAG AA (4.5:1 para texto normal, 3:1 para texto grande)
  • Texto escalable: El contenido permanece legible y funcional cuando se amplía hasta el 200%
  • Indicadores de enfoque claros: Contornos visibles en elementos interactivos al usar navegación por teclado
  • Sin información solo por color: La información importante no se transmite únicamente a través del color

Navegación por teclado

  • Acceso completo por teclado: Todos los elementos interactivos pueden ser accedidos usando solo un teclado
  • Orden lógico de tabulación: Navegación secuencial a través de elementos de la página
  • Atajos de teclado: Los atajos estándar del navegador funcionan en todo el sitio
  • Sin trampas de teclado: Los usuarios pueden navegar alejándose de cualquier elemento usando métodos estándar

Contenido y medios

  • Texto alternativo: Texto alternativo descriptivo para todas las imágenes informativas
  • Texto de enlace significativo: Enlaces que describen su destino o función
  • Etiquetas de formulario: Etiquetas claras asociadas con todas las entradas de formulario
  • Mensajes de error: Mensajes claros y útiles cuando falla la validación del formulario

Soporte multilingüe

  • 13 idiomas soportados: Incluyendo soporte de derecha a izquierda (RTL) para árabe
  • Atributos de idioma apropiados: Códigos de idioma correctamente marcados para tecnologías asistivas
  • Adaptaciones culturales: Formato apropiado para números, fechas y dirección del texto

Compatibilidad con tecnología asistiva

Nuestro sitio web ha sido probado con:

  • Lectores de pantalla: NVDA, JAWS, VoiceOver, TalkBack
  • Software de control por voz: Dragon NaturallySpeaking, Control por voz
  • Navegación solo por teclado: Para usuarios que no pueden usar un dispositivo señalador
  • Zoom del navegador: Hasta 200% de magnificación sin desplazamiento horizontal

Cumplimiento de estándares

Cumplimiento WCAG 2.1 AA

  • Perceptible: La información y los componentes de la interfaz de usuario se presentan de maneras que los usuarios pueden percibir
  • Operable: Los componentes de la interfaz y la navegación son operables por todos los usuarios
  • Comprensible: La información y la operación de la interfaz de usuario son comprensibles
  • Robusto: El contenido es lo suficientemente robusto para ser interpretado por una amplia variedad de agentes de usuario y tecnologías asistivas

Pruebas y validación

Probamos regularmente nuestro sitio web usando:

  • Herramientas de prueba de accesibilidad automatizadas
  • Pruebas manuales con tecnologías asistivas
  • Pruebas de usuario con personas que tienen discapacidades
  • Validación de código para marcado HTML semántico

Problemas conocidos y limitaciones

Estamos trabajando activamente para abordar cualquier barrera de accesibilidad. Problemas conocidos actualmente:

  • Algunos elementos interactivos complejos pueden requerir pruebas adicionales con tecnologías asistivas específicas
  • El contenido de terceros (como videos incrustados) puede no cumplir completamente con nuestros estándares de accesibilidad

Comentarios y contacto

Agradecemos los comentarios sobre la accesibilidad de nuestro sitio web. Si encuentra alguna barrera de accesibilidad o tiene sugerencias para mejorar:

Al contactarnos, incluya por favor:

  • La página específica o característica con la que tiene problemas
  • La tecnología asistiva que está usando (si aplica)
  • Una descripción de la barrera de accesibilidad
  • Su información de contacto para seguimiento

Tiempo de respuesta

Nuestro objetivo es responder a los comentarios de accesibilidad dentro de 2 días hábiles y trabajaremos para resolver los problemas lo más rápido posible. Los problemas complejos pueden requerir tiempo adicional para pruebas e implementación adecuadas.

Mejora continua

La accesibilidad es un esfuerzo continuo. Regularmente:

  • Revisamos y actualizamos nuestras políticas de accesibilidad
  • Capacitamos a nuestro equipo de desarrollo en mejores prácticas de accesibilidad
  • Realizamos pruebas de usuario con personas que tienen discapacidades
  • Monitoreamos estándares y tecnologías de accesibilidad emergentes

Formatos alternativos

Si necesita información de nuestro sitio web en un formato alternativo, contáctenos. Podemos proporcionar contenido en:

  • Letra grande
  • Formato de audio
  • Lenguaje simplificado
  • Otros formatos bajo petición

Contenido de terceros

Aunque nos esforzamos por asegurar que todo el contenido en nuestro sitio web sea accesible, algún contenido de terceros puede no cumplir completamente con nuestros estándares de accesibilidad. Trabajamos con nuestros socios para mejorar la accesibilidad y estamos felices de ayudar a los usuarios a acceder a este contenido de maneras alternativas.

Información legal

Esta declaración de accesibilidad se aplica al sitio web de Divinci AI (divinci.ai y todos los subdominios). Revisamos y actualizamos esta declaración regularmente para reflejar nuestros esfuerzos actuales de accesibilidad y cualquier cambio en nuestro sitio web.

Última actualización: 20 de enero de 2025

Para preguntas sobre esta declaración de accesibilidad o para reportar problemas de accesibilidad, contacte a nuestro equipo de accesibilidad usando la información proporcionada anteriormente.
