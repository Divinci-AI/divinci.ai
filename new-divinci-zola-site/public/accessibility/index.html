Accessibility Statement

Accessibility Statement

Divinci AI is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Our Commitment

We are dedicated to providing an inclusive experience for all users, regardless of ability or technology. Our website aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 at the AA level.

Accessibility Features

Navigation and Structure

  • Skip Navigation Links: Quick access to main content areas
  • Clear Heading Structure: Logical hierarchy from H1 to H6 for screen readers
  • Landmark Regions: Properly marked navigation, main content, and footer areas
  • Consistent Navigation: Predictable layout and menu structure across all pages

Visual Design

  • High Contrast: Color combinations that meet WCAG AA standards (4.5:1 for normal text, 3:1 for large text)
  • Scalable Text: Content remains readable and functional when zoomed up to 200%
  • Clear Focus Indicators: Visible outlines on interactive elements when using keyboard navigation
  • No Color-Only Information: Important information is not conveyed through color alone

Keyboard Navigation

  • Full Keyboard Access: All interactive elements can be accessed using only a keyboard
  • Logical Tab Order: Sequential navigation through page elements
  • Keyboard Shortcuts: Standard browser shortcuts work throughout the site
  • No Keyboard Traps: Users can navigate away from any element using standard methods

Content and Media

  • Alternative Text: Descriptive alt text for all informative images
  • Meaningful Link Text: Links that describe their destination or function
  • Form Labels: Clear labels associated with all form inputs
  • Error Messages: Clear, helpful messages when form validation fails

Multilingual Support

  • 13 Languages Supported: Including right-to-left (RTL) support for Arabic
  • Proper Language Attributes: Correctly marked language codes for assistive technologies
  • Cultural Adaptations: Appropriate formatting for numbers, dates, and text direction

Assistive Technology Compatibility

Our website has been tested with:

  • Screen Readers: NVDA, JAWS, VoiceOver, TalkBack
  • Voice Control Software: Dragon NaturallySpeaking, Voice Control
  • Keyboard-Only Navigation: For users who cannot use a pointing device
  • Browser Zoom: Up to 200% magnification without horizontal scrolling

Standards Compliance

WCAG 2.1 AA Compliance

  • Perceivable: Information and UI components are presentable in ways users can perceive
  • Operable: Interface components and navigation are operable by all users
  • Understandable: Information and UI operation are understandable
  • Robust: Content is robust enough to be interpreted by a wide variety of user agents and assistive technologies

Testing and Validation

We regularly test our website using:

  • Automated accessibility testing tools
  • Manual testing with assistive technologies
  • User testing with individuals who have disabilities
  • Code validation for semantic HTML markup

Known Issues and Limitations

We are actively working to address any accessibility barriers. Currently known issues:

  • Some complex interactive elements may require additional testing with specific assistive technologies
  • Third-party content (such as embedded videos) may not fully conform to our accessibility standards

Feedback and Contact

We welcome feedback on the accessibility of our website. If you encounter any accessibility barriers or have suggestions for improvement:

Email: accessibility@divinci.ai
Phone: +1 (555) 123-4567

When contacting us, please include:

  • The specific page or feature you’re having trouble with
  • The assistive technology you’re using (if applicable)
  • A description of the accessibility barrier
  • Your contact information for follow-up

Response Time

We aim to respond to accessibility feedback within 2 business days and will work to resolve issues as quickly as possible. Complex issues may require additional time for proper testing and implementation.

Continuous Improvement

Accessibility is an ongoing effort. We regularly:

  • Review and update our accessibility policies
  • Train our development team on accessibility best practices
  • Conduct user testing with individuals who have disabilities
  • Monitor emerging accessibility standards and technologies

Alternative Formats

If you need information from our website in an alternative format, please contact us. We can provide content in:

  • Large print
  • Audio format
  • Simplified language
  • Other formats upon request

Third-Party Content

While we strive to ensure all content on our website is accessible, some third-party content may not fully conform to our accessibility standards. We work with our partners to improve accessibility and are happy to help users access this content in alternative ways.

Legal Information

This accessibility statement applies to the Divinci AI website (divinci.ai and all subdomains). We review and update this statement regularly to reflect our current accessibility efforts and any changes to our website.

Last updated: January 20, 2025

For questions about this accessibility statement or to report accessibility issues, please contact our accessibility team using the information provided above.
