{{ translations.support.faq.questions.training_time.question | default(value="How long does the AI training process take?") }}

Training duration depends on the volume and complexity of your data:

  • Basic models (under 100MB of data): 30-60 minutes
  • Standard models (100MB-1GB): 2-4 hours
  • Comprehensive models (1GB-5GB): 4-12 hours
  • Enterprise models (5GB+): Custom timeframes

Our platform provides real-time progress updates and notifications when training is complete. You can continue using the platform during training, and incremental training updates typically take less time than initial training.

{{ translations.support.faq.questions.integration.question | default(value="Can I integrate my Divinci AI with existing systems?") }}

Yes, Divinci AI provides multiple integration options:

  • API Integration: RESTful API for custom applications
  • Web Widgets: Embeddable chat interfaces for websites
  • Mobile SDK: Native integration for iOS and Android apps
  • Pre-built Connectors: For platforms like Slack, Microsoft Teams, Zendesk, and Salesforce
  • Webhook Support: For event-based integration flows

Our Developer Hub provides detailed documentation, code samples, and integration guides for all supported platforms.

{{ translations.support.faq.questions.security.question | default(value="How secure is my data with Divinci AI?") }}

Divinci AI implements robust security measures including:

  • Data Encryption: All data is encrypted in transit and at rest
  • Role-Based Access: Granular permission controls
  • SOC 2 Compliance: Regular security audits
  • Data Isolation: Your training data is used only for your AI models
  • Private Cloud Options: For enterprise customers with enhanced security requirements

For more information, please review our Privacy Policy and Security Documentation.
