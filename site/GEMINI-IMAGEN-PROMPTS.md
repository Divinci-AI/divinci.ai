## Usage Guide

### Best Practices:
1. **Aspect Ratios:**
   - Blog hero images: 16:9 (1920x1080)
   - Social media: 1:1 (1080x1080)
   - Feature highlights: 4:3 (1600x1200)

2. **Style Consistency:**
   - Always mention "Renaissance" + "modern tech" blend
   - Include specific color palette from Divinci AI brand
   - Request "photorealistic" or specific art style

3. **Prompt Structure:**
   ```
   [Main subject] + [Style reference] + [Key details] + [Color palette] + [Lighting/mood] + [Quality/aesthetic]
   ```

4. **Iteration Tips:**
   - If first result is too modern: Add "classical Renaissance painting style"
   - If too classical: Add "subtle futuristic elements" or "modern tech overlays"
   - For more detail: Add "highly detailed", "intricate", "photorealistic"
   - For cleaner look: Add "clean", "minimalist Renaissance", "elegant composition"

5. **After Generation:**
   - Download as PNG or JPG (highest quality)
   - Save to `static/images/` directory
   - Upload to R2 for production use
   - Update blog post or page with new image URL

### Imagen 3 Specific Tips:
- Keep prompts under 400 words
- Be specific about composition and framing
- Mention specific Renaissance artists for style reference (da Vinci, Michelangelo, Raphael)
- Include lighting descriptions for mood
- Specify "photorealistic" vs "illustration" vs "painting" style
- Use concrete visual elements rather than abstract concepts

### Quick Prompt Template:
```
[Subject doing action], [Renaissance artist] style meets [modern element], [specific details], [color palette: ochre, green, gold], [lighting type], [quality: photorealistic/illustration], [composition: close-up/wide shot/aerial]
```
