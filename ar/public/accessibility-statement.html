Our Commitment to Accessibility

Divinci.ai is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone, and applying the relevant accessibility standards.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

Divinci.ai is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Accessibility Features

Divinci.ai includes the following accessibility features:

  • Skip links to bypass navigation and go directly to main content
  • Keyboard accessibility for all interactive elements
  • Sufficient color contrast for text and interactive elements
  • Text alternatives for non-text content
  • Proper heading structure and landmark regions
  • Form labels and instructions
  • Focus indicators for keyboard navigation
  • Responsive design for different screen sizes
  • ARIA attributes for enhanced screen reader support
  • Keyboard shortcuts for common actions

Known Limitations

Despite our best efforts to ensure accessibility of Divinci.ai, there may be some limitations. Below is a list of known limitations:

  • Some older content may not be fully accessible
  • Some third-party content may not be fully accessible
  • Some dynamic content may not be announced properly to screen readers
  • Some complex interactive elements may have limited keyboard support

We are working to address these issues and improve the accessibility of our website.

Assessment Approach

Divinci.ai assessed the accessibility of this website by the following approaches:

  • Self-evaluation using automated testing tools
  • Manual testing with assistive technologies
  • User testing with people who use assistive technologies

Compatibility with Browsers and Assistive Technology

Divinci.ai is designed to be compatible with the following assistive technologies:

  • Screen readers (including NVDA, JAWS, and VoiceOver)
  • Screen magnifiers
  • Speech recognition software
  • Keyboard-only navigation

Divinci.ai is compatible with the following browsers:

  • Google Chrome (latest version)
  • Mozilla Firefox (latest version)
  • Apple Safari (latest version)
  • Microsoft Edge (latest version)

Technical Specifications

Accessibility of Divinci.ai relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

These technologies are relied upon for conformance with the accessibility standards used.

Accessibility Improvements

We are continuously working to improve the accessibility of Divinci.ai. Recent improvements include:

  • Added skip links for keyboard navigation
  • Improved color contrast for text and interactive elements
  • Added proper labels to all form controls
  • Enhanced keyboard navigation and focus management
  • Added alternative text to all informative images
  • Improved document structure with proper headings and landmarks
  • Added ARIA attributes for enhanced screen reader support
