Accessibility Improvements

This page demonstrates various accessibility improvements implemented on the website.

A skip link has been added to the top of the page, allowing keyboard users to skip directly to the main content. Try pressing Tab when the page first loads.

All interactive elements are now keyboard accessible. Try navigating the page using only the Tab key.
