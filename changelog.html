v1.0.0

Initial Public Release

May 1, 2025
New

Our first public release of Divinci AI after extensive private beta testing. This release includes all core functionality that allows users to create custom AI solutions.

  • Custom AI model creation with no-code interface
  • Advanced RAG (Retrieval-Augmented Generation) capabilities
  • Document processing and knowledge extraction
  • Vector embedding optimization
  • API access for enterprise integrations
  • Quality assurance tooling for LLMs
  • Release cycle management for AI projects

New Improvement

Final pre-release version with completed feature set and significant improvements based on beta user feedback.

  • Enhanced user interface with accessibility improvements
  • Optimized backend performance for faster model training and response times
