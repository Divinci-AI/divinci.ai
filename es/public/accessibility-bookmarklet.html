Accessibility Bookmarklet

This bookmarklet adds accessibility improvements to any webpage. Drag the button below to your bookmarks bar, then click it on any page to apply accessibility enhancements.

Make Accessible

How to Use

  1. Drag the "Make Accessible" button above to your bookmarks bar.
  2. Navigate to any webpage you want to make more accessible.
  3. Click the bookmarklet in your bookmarks bar.
  4. The page will be enhanced with accessibility improvements.

Accessibility Improvements

This bookmarklet adds the following accessibility improvements to webpages:

  • Skip Link: Adds a skip link at the top of the page for keyboard users.
  • Keyboard Navigation: Makes all interactive elements keyboard accessible.
  • Color Contrast: Improves color contrast for better readability.
  • Form Labels: Adds missing labels to form controls.
  • Image Alt Text: Adds alternative text to images.
  • Document Structure: Improves the semantic structure of the page.
  • Focus Indicators: Adds visible focus indicators for keyboard users.
  • ARIA Attributes: Adds missing ARIA attributes for screen readers.
  • Keyboard Shortcuts: Adds keyboard shortcuts for common actions.

Note: While this bookmarklet can improve accessibility, it may not fix all issues. For the best results, accessibility should be built into the website from the beginning.
